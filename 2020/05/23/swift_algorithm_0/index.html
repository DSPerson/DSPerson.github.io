数组

数组插入和删除，都要重新移动位置内存位置要重新布局，相较于它的查询较慢.

插入和删除时间复杂度O(N), 如果很幸运的插入到最后面复杂度是O(1), 但是插入到最前面则是O(N), 平均是O(N/2)还是O(N)

查询时间复杂度O(1)
