lldb 学习

发表于 2017-06-14   |   分类于 iOS功能   |  

lldb 替代gdb 在程序运行中 暂停 即可使用lldb了 在项目中可以有多帮助

安装Chisel

Chisel网址

也可以 在终端直接

brew update
brew install chisel

成功后 将红色框内在 终端 中 输入 touch ~/.lldbinit 复制进去 重启Xcode

常用命令

都是在项目在运行后 暂停条件下

使用 visualize + 内存地址即可在预览中打开图片查看

使用 pvc 查看当前NavigationBar栈

使用 pintnals + 内存地址 查看 私有属性

使用 pviews 从当前到当前View所有视图 pviews + 内存地址

使用 caflush 手动刷新

使用 presponder + 内存地址 查看当前地址的层级

使用 taplog 会自动关闭暂停之后点击任意位置 会打印出最近一个 响应点击时间的View

使用 pclass + 内存地址 查看继承关系

使用 bmessage + 类 + 方法 类似断点

使用 show 或者 hide + 内存地址 来 展示或者隐藏控件
