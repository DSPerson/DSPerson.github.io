Mac 上搭建Mysql总结

发表于 2017-01-19 |

使用Homebrew 搭建Mysql是非常稳妥，因为墙的原因可能速度较慢一些，如果使用梯子可能会很快(比如我)。下面看到我写分号的你也得记得写上。

第一步执行

brew install mysql

之后就是等待了建议翻墙速度汇款很多因为内容较大 400多M

启动服务器

base mysql.server start

第二步 进入Mysql

mysql -uroot -p

此处会让你输入密码 记得 直接回车！！！直接回车！！！直接回车！！！我在这个地方耽误不少时间

此处应该有一个成功的图片
/-------------------\

查看当前数据库

show databases;

第三步设置密码
此处如果直接设置会提示你选用那个数据库 可以用命令行创建一个自己的数据库不要动它自带的数据库，以免出现错误。

先创建一个新的数据库

create databases DS;

如果不想错的话就把分号加上吧 当然你复制下来最好DS数据库名字随便改别重复就行。。。

use DS;

设置密码

set password=password('xxxxxx')

xxxx即密码
